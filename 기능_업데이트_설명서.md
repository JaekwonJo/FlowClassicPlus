# 기능 업데이트 설명서

- 마지막 업데이트: 2026-03-10
- 대상: `Flow Classic Plus`

## 1. 지금 실제로 잘 되는 기능
- 프롬프트 자동화
- S001~S### 자동 반복
- 다운로드 자동화
- 생성 옵션 자동찾기/테스트
  - 프롬프트 자동화용
  - S자동화용

## 2. 최근 반영된 핵심 기능
### 2-1. Classic Plus 전용 메인 메뉴
- 원본 클래식과 헷갈리지 않게 큰 아이콘 첫 화면 추가
- 메인 메뉴 복귀 버튼 추가

### 2-2. 타이핑 속도 슬라이더
- x1 ~ x20으로 선택 가능
- 고정 속도가 아니라 ±30% 랜덤 변화 적용
- 글자 입력 속도, 행동 딜레이, 생각 시간에 같이 반영

### 2-3. 생성 옵션 도구 분리
- 프롬프트 자동화용 생성 옵션 자동찾기/테스트
- S자동화용 생성 옵션 자동찾기/테스트
- 서로 다른 저장값을 사용하도록 분리

### 2-4. 봇 작업창 열기 버튼
- 사용자가 봇 전용 작업창을 먼저 띄워서 상태를 맞출 수 있게 함

### 2-5. 입력방식 저장 보강
- `typing / paste / mixed`가 비정상 fallback으로 다시 `paste`로 떨어지지 않도록 보강

### 2-6. 번호 이동 입력칸 가독성 개선
- 배경/글자색을 다크 테마에 맞게 정리

### 2-7. 다운로드 실패 대기시간/완료 요약
- 다운로드 자동화에 `실패 판단 대기(초)` 설정 추가
- 재시도 없이, 설정한 시간 안에 못 찾으면 다음 항목으로 이동
- 성공/실패 로그를 더 짧게 정리
- 작업 완료 시 메타데이터 팝업 표시
- 완료 요약을 `logs/completion_summary_날짜_시간.txt` 로 저장

## 3. 지금 헷갈릴 수 있는 부분 설명
### `모드`
- 이번 실행에서 최종적으로 이미지로 만들지, 영상으로 만들지 정하는 값

### `기본값 이미지 맞춤 완료` / `기본값 동영상 맞춤 완료`
- 봇 작업창의 시작 상태를 사용자가 한 번 맞춰줬다는 확인 버튼
- 출발점 확인용 버튼이지, 이번 작업 목표를 고르는 버튼이 아님

## 4. 지금 안 쓰는 것 / 단순화된 것
- 생성 옵션에서 가로/세로, x1/x2/x3/x4는 현재 핵심 흐름에서 제외
- 우선순위는 Image / Video 모드 전환과 핵심 자동화 기능 안정화
