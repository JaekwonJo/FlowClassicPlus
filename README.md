# Flow Classic Plus

원본 클래식을 건드리지 않고, 실제로 쓰는 기능만 하나씩 개선해 나가는 버전입니다.

## 현재 상태
- 프롬프트 자동화: 정상
- S001~S### 자동 반복: 정상
- 다운로드 자동화: 정상
- 생성 옵션 자동찾기/테스트: 정상

## 핵심 원칙
- 원본 클래식 보존
- Classic Plus에서만 개선
- 기능부터 먼저 안정화
- 큰 구조 변경은 천천히
- 이어달리기는 맨 마지막

## 지금 중요한 개념
### 프롬프트 자동화
- 기본 기준 상태: 이미지
- `모드`: 이번 실행 목표(Image / Video)
- `기본값 이미지 맞춤 완료`: 시작 전 출발 상태 확인

### S자동화
- 기본 기준 상태: 동영상
- 시작 전 `기본값 동영상 맞춤 완료` 확인 후 진행

## 바로 실행
1. `1_필수라이브러리_설치.bat`
2. `2_오토_프로그램_실행.bat`
