# Flow Classic Plus

원본 클래식을 건드리지 않고, 실제로 쓰는 기능만 하나씩 개선해 나가는 버전입니다.

## 2026년 3월 12일 완성품 기준
- 현재 GitHub 기준 완성 스냅샷: `2026년 3월 12일 완성품`
- 프롬프트 자동화 / S자동화 / 다운로드 / 이어달리기 핵심 흐름이 실사용 성공한 기준 버전
- 노트북에서 내려받아 시험하기 좋은 기준점
- 이후 남은 큰 작업은 `디자인/배치 정리`와 `잘 쓰지 않는 고급 설정 정리`

## 현재 상태
- 프롬프트 자동화: 정상
- S001~S### 자동 반복: 정상
- 다운로드 자동화: 정상
- 생성 옵션 자동찾기/테스트: 정상
- 이어달리기 핵심 흐름: 정상
- 데스크탑 / 노트북 둘 다 핵심 흐름 통과 확인

## 핵심 원칙
- 원본 클래식 보존
- Classic Plus에서만 개선
- 기능부터 먼저 안정화
- 큰 구조 변경은 천천히
- 이어달리기는 맨 마지막

## 지금 중요한 개념
### 프롬프트 자동화
- 기본 기준 상태: 이미지
- `모드`: 이번 실행 목표(Image / Video)
- `기본값 이미지 맞춤 완료`: 시작 전 출발 상태 확인

### S자동화
- 기본 기준 상태: 동영상
- 시작 전 `기본값 동영상 맞춤 완료` 확인 후 진행

## 바로 실행
1. `1_필수라이브러리_설치.bat`
2. `2_오토_프로그램_실행.bat`

## 노트북에서 다운로드 후 실행 순서
1. `git clone git@github.com:JaekwonJo/FlowClassicPlus.git`
2. `cd FlowClassicPlus`
3. `git checkout complete-2026-03-12`
4. 처음 1회만 `1_필수라이브러리_설치.bat` 실행
5. 그 다음 `2_오토_프로그램_실행.bat` 실행
6. 테스트할 때는 `Flow Classic Plus` 기준으로 확인

## 바탕화면 바로가기 만들기
1. 프로젝트 폴더에서 `바탕화면_바로가기_생성.bat` 실행
2. 바탕화면에 `Flow Classic Plus.lnk` 생성 확인
3. 다음부터는 그 바로가기로 바로 실행
