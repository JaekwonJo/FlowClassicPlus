## 1. 현재 안정 상태
- 원본 클래식은 건드리지 않고 보존
- 실제 개발판은 `Flow Classic Plus`
- 사용자 확인 완료 기능:
  - 프롬프트 자동화 정상
  - S001~S### 자동 반복 정상
  - 다운로드 자동화 정상
  - 프롬프트 생성 옵션 자동찾기/테스트 정상
  - S 생성 옵션 자동찾기/테스트 정상

## 2. 최근 사용자 확인 완료 사항
- 프롬프트 자동화: 실제 입력/제출 성공
- S자동화: 실제 에셋 탐색/입력 성공
- 다운로드 자동화: 실제 다운로드 성공
- 생성 옵션은 프롬프트용과 S용을 분리함

## 3. 현재 가장 중요한 고정 규칙
1. 원본 클래식은 보존
2. 개선 작업은 `Flow Classic Plus`에서만 진행
3. 기능은 클래식 기반 유지
4. 너무 큰 구조 변경 금지
5. 이어달리기는 맨 마지막 작업

## 4. 생성 옵션 관련 현재 기준
### 프롬프트 자동화
- 기본 기준 상태: 이미지
- `모드`는 이번 실행 목표(Image / Video)
- `기본값 이미지 맞춤 완료`는 출발 상태 확인 버튼

### S자동화
- 기본 기준 상태: 동영상
- S자동화는 영상 작업 기준
- `기본값 동영상 맞춤 완료` 확인 후 시작

### 저장/탐색 구조
- 프롬프트용 생성 옵션과 S용 생성 옵션을 분리
- 자동찾기/테스트 로그도 분리 저장

## 5. 최근 중요 커밋
- `76e3bdd` 입력방식 저장 fallback 보강, 번호 이동 칸 가독성 개선
