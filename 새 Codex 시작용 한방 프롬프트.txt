[최근 중요 안정화 포인트]
- 워커 3종(이미지/S자동화/다운로드)은 이제 메인 카드에서 중간 런처 없이 바로 실제 미니 워커 창으로 열립니다.
- 각 워커는 별도 설정 파일/브라우저 프로필 구조이며, 기본 추천 이름 그대로 쓰면 서로 안 섞이게 동작합니다.
- 이어달리기 자동 재시도는 마지막 실패 번호를 기준으로 `프롬프트 -> S자동화 -> 다운로드` 전체를 한 번 더 돌고, 그래도 남으면 마지막으로 `다운로드 자동화 / 이미지 / 1K`를 한 번 더 실행합니다.
- 다운로드 자동화는 없는 S번호를 실제 없는 것으로 빠르게 실패 처리해야 합니다.
- 다운로드 자동화는 상세 화면의 큰 대표 타일에서도 `... -> 다운로드`가 정상으로 작동해야 합니다.
- 1080P/4K는 `업스케일링 중` / `업스케일링 완료`를 구분해 완료 감지 후 바로 다음 번호로 넘어가야 합니다.
- 720P는 업스케일 감시를 타면 안 되고 즉시 다운로드 경로로 처리해야 합니다.
- inline 레퍼런스 첨부는 현재 `@ -> 오래된 순 -> 태그 입력 -> Enter`의 단순 흐름이 안정판입니다. 불필요한 클릭 복귀를 넣지 마세요.
- 프롬프트 파일은 `S011`, `S069::...` 같은 실제 번호 태그를 읽을 수 있습니다.
