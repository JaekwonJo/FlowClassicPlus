# Flow Veo 배포 가이드 (2026-03-03 Ver.01)

## 1) 설치 마법사(.exe) 만들기
1. Windows PC에서 `release/build_installer.bat` 실행
2. 완료되면 `dist/FlowVeo_20260303_Ver01_Setup.exe` 생성

## 2) 최종 사용자 설치/실행
1. `FlowVeo_20260303_Ver01_Setup.exe` 실행
2. 설치 후 바탕화면 또는 시작 메뉴의 `Flow Veo 자동화 봇` 실행
3. 첫 실행에서 자동으로 Python/필수 패키지/Chromium 준비
4. 이후부터는 아이콘 클릭만으로 실행

## 3) 버전 표기
- 앱 타이틀/설치파일 버전: `2026-03-03 Ver.01`

## 4) 주의사항
- 첫 실행은 인터넷 연결 필요
- 보안 프로그램이 다운로드를 막으면 예외 허용 필요
