# 릴리스 노트 - 2026-03-05 FINAL

## 버전
- `v2026.03.05-final`

## 릴리스 성격
- 실사용 안정판 고정 버전
- 핵심 자동화 기능 + 다운로드 자동화 기능 동작 확인 완료

## 포함 기능
1. 프롬프트 자동화
- selector 자동찾기/테스트
- 일시정지/재개/완전중지

2. S반복 자동화
- 시작/끝 번호 범위 실행
- `S01` 형태 0채움 지원
- 에셋 검색 단계 자동화

3. 다운로드 자동화 (분리 모듈)
- 다운로드 자동화 시작 버튼 별도
- 영상/이미지 모드 분리
- 영상 품질: `720P`, `1080P`, `4K`
- 이미지 품질: `1K`, `2K`, `4K`
- selector 자동찾기/테스트(이미지/영상 각각)
- 실패 시 재시도 없이 다음 태그로 진행

## 안정화/보정 사항
1. 동적 UI 대응
- ref 고정 사용 제거, 단계별 재탐색

2. 검색칸 탐색 안정화
- 상단 중앙 검색바 우선 탐지
- 잘못된 입력칸(프로젝트명 등) 오탐 감소

3. 필터/메뉴 안정화
- 좌측 사이드바 필터 우선
- 다운로드 메뉴 재시도 로직 추가
- 이미지 4K 선택 시 수직 이동 경로 + 재시도 적용

4. 다운로드 대기시간 개선
- 1080P/4K 고품질 대기시간 확장

5. 저장/로그 강화
- 다운로드 저장 폴더 설정 UI 추가
- 저장된 전체 파일 경로 로그 출력
- 전용 로그: `download_trace_*.log`
- 전용 리포트: `download_report_*.json`
